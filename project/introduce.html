머신 러닝

머신 러닝의 마법

M1 칩의 머신 러닝 기술이 Mac 안에 탑재된 가능성의 세계를 엽니다. 핵심은 머신 러닝 기술을 활용해 방대한 데이터셋 기반으로 모델을 구축하고 적용할 수 있다는 점입니다.

16코어 Neural Engine

M1 칩에 내장된 16코어 Neural Engine은 초당 최대 11조 회의 연산을 수행합니다. Pro AI 앱으로 음성 인식, 스피치 분석, 스마트 기능이 가능해집니다.

머신 러닝 최적화

CPU, GPU, Neural Engine을 조화롭게 구성하여 각 ML 작업을 최적화합니다. 실시간 데이터 분석에 탁월합니다.

ML 가속장치

고속 ML 전용 가속 장치는 초당 최대 1조 회 연산을 수행하며, 모든 작업이 디바이스 내에서 처리되므로 개인 정보도 보호됩니다.
