## 🙆‍♀️ 프로젝트 기획 의도
러닝 인구 1,000만 시대를 넘어선 현재,  
러닝에 대한 관심은 높아졌지만 **올바르고 안전한 정보는 여전히 부족한 상황**입니다.

본 프로젝트는  
- 러닝의 중요성을 알리고  
- 러너들에게 안전한 러닝 정보와 추천 콘텐츠를 제공하여  
- 보다 **효율적이고 즐거운 러닝 문화**를 만드는 것을 목표로 기획되었습니다.
